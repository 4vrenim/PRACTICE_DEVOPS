# Sử dụng Terraform để triển khai một ứng dụng web với Load Balancer và Auto Scaling Group trên AWS
## Thay thế với Azure
* Tạo file main.tf
* Thực thi Terraform
>terraform init  
>terraform plan  
>terraform apply -auto-approve
